◈ DataLens — Help & Reference
Compare & validate CSV/Excel files · Tested on 5 GB+ files with 13 M+ rows
Compare Mode
Finds row-level differences between a Reference and a Candidate file. Rows are matched by key column value — not position — so reordered rows are handled correctly.
Added rowsRows present in Candidate but not in Reference
Removed rowsRows present in Reference but missing from Candidate
Modified rowsSame key, one or more column values changed
Formatting onlyChange is purely case or whitespace (flagged separately)
Validate Mode
Checks data quality on both files independently. Findings are grouped by severity:
INFOInformational — no action required
WARNINGPotential issue worth reviewing
ERRORDefinite data quality problem
CRITICALBlocking issue — upload not recommended
Key Columns
The unique identifier used to match rows across files. Leave blank to auto-detect (first column that is 100% unique is chosen).
Single keyemployee_id
Composite keydept_id,employee_id
TipIf no unique column exists, comparison still runs but duplicate-key warnings appear
Column Validation Rules
Per-column constraints checked in Validate mode. Add rules under the Validation Rules panel.
RequiredColumn must have no nulls/blanks
Max null rate% of nulls allowed before flagging as error
Expected typeAll values must cast to Int64 / Float64 / Boolean / Date…
Min / Max valueNumeric range check
Regex patternEvery value must match the pattern
Allowed valuesComma-separated whitelist of valid values
File Formats
CSV / TSV / TXTAuto-detects delimiter, encoding, and BOM. Falls back to cp1252 if needed.
Excel (.xlsx)Sheet picker shown when multiple sheets exist. Selected sheet is converted to a temp CSV before processing — no data left on disk.
Large filesFiles are read from disk via streaming (Polars LazyFrame). Nothing is uploaded to the browser. Tested at 5 GB / 13 M rows.
Memory guardYellow banner = caution, Red = blocked. Threshold based on available system RAM.
Outputs & Exports
All exports are written to the exports/ folder inside the app directory. Files are never overwritten — each run adds a datetime suffix.
HTML reportFull comparison report with validation summary and profile
Diff CSVRow-level differences — download via the Export CSV button in results
Diff ExcelSame as CSV but formatted as .xlsx with colour coding
Diff JSONMachine-readable diff for downstream pipelines
